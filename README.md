# User Management Dashboard

A simple and responsive **User Management Dashboard** built with **React**, **Bootstrap 5**, and **Framer Motion**.  
This project allows you to **create, view, edit, and delete users**, with features like **search, sorting, and pagination**.

---

## Features

- **Add / Edit Users** with a simple form.
- **View User Details** on a separate page.
- **Delete Users** with confirmation prompt.
- **Search Users** by name or email.
- **Sort Users** by Name, Email, or Phone.
- **Pagination** to navigate large user lists.
- Responsive layout with **Bootstrap 5**.
- Smooth animations using **Framer Motion**.
- Role-based structure ready for further extensions.

---

## Tech Stack

- **Frontend:** React, Bootstrap 5, Framer Motion
- **Routing:** React Router DOM
- **API Handling:** Axios or custom API functions
- **Icons:** Bootstrap Icons
- **Optional Backend:** JSONPlaceholder (mock API) or any REST API
